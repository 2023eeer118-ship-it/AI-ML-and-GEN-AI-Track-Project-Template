Smart Lender - Project Executable Files Guide

1. Prerequisites
- Python 3.8 or higher installed on the system.
- Modern Web Browser (Google Chrome).

2. How to Run the Server
- Open the terminal/command prompt inside the main application folder.
- Run the server file by typing the command:
  python app.py
- Verify that the terminal logs say: * Running on http://127.0.0.1:5000

3. Accessing the Application
- Open Google Chrome and visit: http://127.0.0.1:5000
- Fill out the form fields and submit to test the XG Boost prediction pipeline.
